This device provides the pipboy2000 local network
To configure this device:
  • Log into the configuration page located at 192.168.0.254 (username:admin, password:admin)
  • Upgrade the software using this file: wr702nv1-us-up.bin which I obtained from the tplink support web-site.
  • Set the working mode to Access point
  • disable security
  • set the SSID=pipboy2000
  • Connect the raspberry pi to the rj45 (wired) port
  • Power up the raspberry pi and wait a couple minutes for it to boot up.
You should now be able to connect a laptop to the pipboy2000 network and browse to page http://192.168.0.100/Pipboy
